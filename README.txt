-------------Tutorial de instalação----------------------
1) Em planejamento>Banco de dados há um arquivo nomedo como beep.sql, esse arquivo deve ser importado na area de importar base de dados do phpMyadmin.

2) Após a importação para o phpMyadmin é necessario entrar no localhost e localizar onde foi extraido a beep

3) apos executar a url vc será redirecionado para o index da beep. Ai é so ser feliz! :)
